# object对象数据类型-普通对象
> {[key]: [value], ...}任何一个对象都是由0到多个銉值对组成的（属性名不能重复）
> 读取、设置属性boj.'属性名' / obj['属性名']
> 数组是特殊的对象类型
>> 数组的属性名（索引，下标）默认生成的是数字，从0开始连续递增
>> 天生默认一个属性名length，存储数组的长度
## 浏览器想执行js代码要：
- 1. 从电脑内存中分配一块内存，用来执行代码（栈内存=>stack）
- 2. 分配一个主线程用来自上而下执行代码
> 栈内存（变量存储空间、值存储空间）
> 每行代码都要经历进栈的过程，执行完后要出栈，出栈后下面的代码才能执行
> let a = 12; // 经历了三个步骤
> 1 创建变量a放到当前栈内存存储空间中
> 2 创建一个值，把它存储到当前栈内存值区域中（简单的基本类型值是这么存储的，复杂的引用数据类型值不是这样存储）
> 3 赋值，把变量名与值关联起来
> 4 复杂值的存储又分成三个步骤：
> 4.1 在内存中分配出一块新内存，用来存储引用类型值（堆内存=> heap）内存中有一个16进㓡地址
> 4.2 把对象的键值对依次存储到堆内存中
> 4.3 把堆内存地址和变量关联起来
> 基本数据类型：按值操作所以也叫作值类型，
> 引用数据类型： 操作的是堆内存的地址，按引用地址操作
## JS中的类型检测
- typeof [val]: 用来检测数据类型的运算符
- instanceof: 用来检测当前实例是否属于某个类
- constructor: 基于构造函数检测数据类型（也是基于类的方式）
- Object.prototype.toString.call(): 检测数据类型最好的办法

* 基于typeof检测出来的结果
* 1. 首先是一个字窜
* 2. 字窜包含对应的类型
* 局限性
* 1. typeof null => 'object' 但是null并不是对象
* 2. typeof [] => 'object'
* 3. typeof /^/ => 'object'
* 4. 基于typeof无法细分出当前值是普通对象还是数组对象，只要是对象类型都会返回'object'

> typeof typeof typeof [] => 'string'
